# Desentupidora-adriano

Landing page de alta conversão para empresa de desentupimento na Grande São Paulo.

## 📋 Sobre o Projeto

Página estática mobile-first com foco em conversão via WhatsApp. Desenvolvida com HTML, CSS e JS puros para máxima velocidade de carregamento e zero dependências.

## 🎨 Design

- **Cores**: Azul escuro corporativo (`#0A1628`) + Verde WhatsApp (`#25D366`)
- **Tipografia**: Inter (Google Fonts)
- **Animações**: Pulse nos CTAs, fade-in com scroll, stagger delays

## 📁 Estrutura

```
Desentupidora-adriano/
├── index.html    # Página principal
├── style.css     # Design system e estilos
├── script.js     # Animações e interatividade
├── .gitignore
└── README.md
```

## ⚙️ Configuração

1. Substitua `5511999999999` pelo número real de WhatsApp em `index.html` e `script.js`.
2. Abra `index.html` no navegador para visualizar localmente.

## 🚀 Deploy

Compatível com Vercel, Netlify, GitHub Pages — qualquer hospedagem estática.

## 📄 Seções

1. Hero (CTA pulsante)
2. Dor / Agitação
3. Serviços (4 cards)
4. Benefícios (checklist)
5. Como Funciona (timeline 3 passos)
6. Footer CTA + botão flutuante WhatsApp
